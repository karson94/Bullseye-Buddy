## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/bullseye_buddy.git
   cd bullseye_buddy
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app locally:**
   ```bash
   flutter run -d chrome
   ```
